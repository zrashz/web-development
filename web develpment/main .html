HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages. It describes the structure of a web page using markup.

Example:

            
                <!DOCTYPE html>
                <html>
                <head>
                    <title>Example</title>
                </head>
                <body>
                    <h1>Hello, World!</h1>
                    <p>This is a paragraph.</p>
                </body>
                </html>

CSS (Cascading Style Sheets)

CSS is used for styling web pages. It allows you to control the layout, colors, fonts, and other visual aspects of a webpage.

Example:

            
                body {
                    background-color: lightblue;
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: navy;
                }

JavaScript

JavaScript is a programming language that enables interactive web pages. It can be used to create dynamic content, handle events, and much more.

Example:

            
                document.getElementById('demo').innerHTML = 'Hello, JavaScript!';
