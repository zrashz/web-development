Question 5:Discuss the advantages and disadvantages of Static vs. Dynamic Web Pages.

Static Web Pages:

Advantages Disadvantages
Effortlessness: Easier to code and maintain due to minimal technical complexities. Constrained interactivity: Offers a passive user experience with minimal engagement features.
Speed: Load faster as content is pre-written and doesn't require real-time generation. Content updates: Updating content requires manual changes to each individual page.
Cost-effective: No need for complex backend infrastructure or ongoing server-side processing. Personalization: Cannot personalize content based on user preferences or behavior.
Security: Less vulnerable to hacking due to the absence of dynamic elements and scripts. Scalability: Difficult to scale for large amounts of content or user traffic.
SEO-friendly: Search engines can easily crawl and index static content. Not ideal for: Complex applications, e-commerce stores, social media platforms, or frequently updated content.

Dynamic Web Pages:

Advantages Disadvantages
Interactivity: Allows for user interaction with forms, comments, live updates, and dynamic content. Complexity: Requires more development and maintenance effort due to server-side scripting and databases.
Personalization: Can personalize content based on user preferences, location, or past behavior. Speed: Can load slower due to real-time content generation and processing.
Scalability: Can easily scale to handle large amounts of content and user traffic. Cost: May require additional resources for server infrastructure and development expertise.
Flexibility: More flexible and adaptable to different site features and functionalities. Security: Increased risk of vulnerabilities due to dynamic elements and scripts.
Ideal for: Complex applications, e-commerce stores, social media platforms, and frequently updated content. SEO: Dynamic content can pose challenges for search engine optimization.
